# 🌌 AI Internship Portfolio Framework & Numerical Physics Engine

A high-performance, dark-mode, production-ready portfolio built to host the structural models and findings compiled during my AI Internship program. This project highlights the confluence of physical dynamics, multivariable numerical algorithms, and deep neural configurations.

## 🎓 Academic Context
*   **Researcher:** Sajal Baruah
*   **Institutional Alignment:** 4th Semester B.Sc. (Physics Major, Mathematics Minor), Gauhati University.
*   **Research Vibe:** High contrast dark glassmorphism integrated with reactive neon cyan (`#00f3ff`) and purple (`#bc13fe`) assets.

---

## 🛠️ Integrated Technological Stack
*   **Client Core Engine:** Semantic HTML5, Modular CSS3 Custom Properties (Glassmorphism Core), Vanilla Functional ECMAScript (JavaScript).
*   **Visual Elements:** Pure CSS 3D Transformation matrices (`preserve-3d`), hardware-accelerated layouts, and Instagram interactive iframe components.
*   **Computational Backend Core:** Python 3, TensorFlow/Keras Dense & Recurrent layer engines, NLTK NLP pipeline configurations, Scikit-Learn Vector models, NumPy matrix engines.
